# Sistema ICA - Formularios Dinámicos

## Contexto
Sistema web para gestión de declaraciones del Impuesto ICA para múltiples municipios en Colombia.

## Objetivo
Construir un sistema de formularios completamente dinámico (config-driven).

## Stack
- React + Vite + TypeScript
- React Hook Form
- Zod
- TailwindCSS

## Principios
- Nada hardcodeado
- Configuración por municipio
- Escalable y mantenible

## UI/UX
- Formulario tipo step-by-step (multi-step wizard)
- Cada paso se construye dinámicamente desde configuración
